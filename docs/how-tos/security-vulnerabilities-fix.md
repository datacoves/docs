# How to run security vulnerabilities check and fix them

## React app

### Install

```shell
yarn add yarn-audit-fix -D
```

### Run

```shell
yarn-audit-fix
```

Learn more: https://yarnpkg.com/package?name=yarn-audit-fix

## Django app

### Install

```shell
pip install pip-audit
```

### Run

```shell
pip-audit -r ./requirements.txt
```
